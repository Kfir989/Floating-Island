Projects

This is the projects section where I showcase my work and accomplishments.

E-Commerce Digital Store

Full-stack e-commerce web application built with React on the frontend and Node.js/Express on the backend. Users can browse a dynamic product catalog, add items to a shopping cart, and simulate checkout. Product data is stored in MongoDB, and the architecture emphasizes responsive design, modular components, and real-time state updates with React hooks. Automated tests are integrated with GitHub Actions. The live deployment showcases an end-to-end workflow from client to server, including secure data connections and frontend-backend communication.

Smart Waste Management Dashboard

A JavaFX-based desktop application that provides real-time monitoring and control of city waste collection. The dashboard offers features such as fill-level tracking of smart bins, salary and work-hour analytics, and route status updates for collection teams. Tech stack: Java, JavaFX, RESTful APIs, and a backend database for storing sensor data and workforce information.

GPS Routing & Optimization Engine

An algorithmic module that calculates optimal collection routes for waste trucks using OSRM (Open Source Routing Machine) and custom heuristics. This engine reduces travel distance and time while ensuring all sensor-enabled bins are serviced efficiently. Tech stack: Java, OSRM for map and routing services, and custom path-finding algorithms.

Floating Island

3D web experience built with Three.js and Vite, showcasing projects and personal information in an immersive, game-like environment. Visitors can explore an animated floating island and interact with different signs to reveal sections such as About Me, Projects, and Overview, creating a dynamic and visually engaging way to present a developer portfolio.
